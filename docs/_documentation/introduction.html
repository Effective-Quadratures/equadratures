Download and installation¶

To download and install the code please use the python package index command:

pip install equadratures

Alternatively you can visit our GitHub page and click either on the Fork Code button or Clone or download. For installation on Mac and Linux systems, simply type the following into the terminal.

sudo python setup.py install

For installation on Windows, you will need Anaconda; select the Python 2.7 option. Upon successful installation, open the Sypder IDE and go into the Effective-Quadratures-master directory and type the above in the command window. This should build the code. Just make sure you include the location of effective_quadratures folder to your python file and you should be good to go. To run this code you will require python 2.7, numpy, scipy and matplotlib.
