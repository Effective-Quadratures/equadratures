# Optimal Quadrature Subsampling

Set of codes for constructing polynomial surrogates using optimal quadrature subsampling both with and without gradients. For details see reference:

Seshadri, P., Narayan, A., Mahadevan, S., (2016) "Optimal Quadrature Subsampling for Least Squares Polynomial Approximations",  arXiv:1601.05470.
